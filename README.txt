Para utilizar la aplicacion solamente se debe ingresar dos datos, en
 la primera seccion (peso) los datos deben ser en kilogramos y la 
segunda seccion la estatura en metros como por ejemplo 1.70m y 
finalmente dar en el boton de calcular para obtener el resultado, con 
el resultado se podra comparar en la imagen superior que aparece con 
una tabla donde se encuentran los rangos tanto para varon como para 
mujer y asi poder saber en que estado se encuentra la persona.

Aplicacion realizada en Android Studio version 1.5.1 
API 26:android 6.0
Sistema operativo Windows 10
